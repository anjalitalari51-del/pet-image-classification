# Pet Image Classification Using Support Vector Machine (SVM)

## Overview
This project is a machine learning application that classifies pet images into two categories: Cats and Dogs. The classification is performed using a Support Vector Machine (SVM) algorithm. The project demonstrates image preprocessing, feature extraction, model training, and prediction techniques in Python.

## Objectives
- Classify pet images as Cat or Dog.
- Train a machine learning model using SVM.
- Evaluate model performance using accuracy metrics.
- Demonstrate image classification concepts in a simple and understandable way.

## Technologies Used
- Python
- OpenCV
- NumPy
- Scikit-learn
- Support Vector Machine (SVM)

## Project Structure

### Source Code
- `svm_train.py` – Trains the SVM model using image data.
- `predict.py` – Predicts the class of a given image.
- `show_prediction.py` – Displays prediction results.
- `check_images.py` – Verifies and validates image files.

### Screenshots
Execution screenshots are provided in the screenshots folder for reference.

## Dataset
The dataset consists of Cat and Dog images organized into separate folders. Images are preprocessed before being used for training and testing.

## Methodology
1. Load image dataset.
2. Resize and preprocess images.
3. Convert images into feature vectors.
4. Split dataset into training and testing sets.
5. Train the SVM classifier.
6. Evaluate model accuracy.
7. Predict classes for new images.

## Results
The trained SVM model successfully classifies pet images and achieves reasonable accuracy on the test dataset.

## Future Improvements
- Increase dataset size for better accuracy.
- Apply feature engineering techniques.
- Implement deep learning models such as CNN.
- Develop a graphical user interface for predictions.

## Conclusion
This project demonstrates how Support Vector Machines can be used for image classification tasks. It provides a simple and effective approach for distinguishing between Cat and Dog images using machine learning techniques.
